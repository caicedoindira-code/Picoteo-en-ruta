# Lenguaje de Marcas - Picoteo en Ruta

## Descripción

Esta carpeta contiene la parte web del proyecto Picoteo en Ruta, una empresa ficticia dedicada al alquiler de stands de comida, bebida y dulces para eventos.

La web está diseñada para promocionar los servicios de la empresa, mostrar los modelos de carritos disponibles y permitir que el usuario solicite una reserva mediante un formulario.

## Archivos incluidos

- `index.html`: estructura principal de la página web.
- `style.css`: estilos visuales de la web.
- Imágenes de los carritos, equipo y logotipo.

## Secciones de la web

La web incluye las siguientes secciones:

- Inicio / Conócenos
- Nuestro equipo
- Modelos de carritos
- Formulario de reserva
- Footer con información de contacto

## Tecnologías utilizadas

- HTML5
- CSS3
- Diseño responsive básico
- Imágenes locales

## Cómo visualizar la web

Para abrir la web, solo hay que ejecutar el archivo:

```text
index.html
```

### Relación con el proyecto

La web representa la parte visible de Picoteo en Ruta. Su objetivo es presentar la empresa, mostrar sus servicios y facilitar que los clientes contacten para reservar un carrito gastronómico para eventos.
